## 🎯 게임 소개

**상식 퀴즈**는 다양한 분야의 지식을 테스트하는 인터랙티브 웹 퀴즈 게임입니다.
각 문제당 10초의 긴장감 넘치는 제한 시간 안에 정답을 맞춰보세요!

### 왜 이 게임을 해야 할까요?

- 🎲 **매번 다른 경험**: 무작위로 선택되는 10개의 문제로 매번 새로운 도전
- ⚡ **빠른 플레이**: 게임 시작 시 모든 문제를 미리 로드하여 끊김 없는 플레이
- 🏆 **11단계 등급 시스템**: 0점부터 10점까지, 당신의 실력을 정확하게 평가
- 💡 **즉각적인 학습**: 틀린 문제도 상세한 해설로 배울 수 있어요
- 🎨 **아름다운 디자인**: 현대적이고 직관적인 UI/UX

## 🎮 게임 플레이

### 1️⃣ 게임 시작
메인 화면에서 "랜덤 퀴즈 시작!" 버튼을 클릭하면 10개의 퀴즈가 즉시 생성됩니다.

### 2️⃣ 문제 풀이
- **제한 시간**: 각 문제당 10초
- **4개의 선택지**: 정답을 고르세요
- **시간 초과**: 자동으로 오답 처리됩니다
- **즉각적인 피드백**: 정답/오답과 함께 상세한 해설 제공

### 3️⃣ 결과 확인
- **정답률**: 당신의 정확도를 확인하세요
- **등급**: 11단계 등급 시스템으로 실력 평가
- **풀이 기록**: 각 문제별 정답/오답 확인

## 🛠️ 기술 스택

### Frontend
- **React 18** - 최신 React 기반 UI
- **TypeScript** - 타입 안전성
- **Tailwind CSS** - 유틸리티 기반 스타일링
- **Vite** - 빠른 빌드 도구

### Backend
- **FastAPI** - Python 기반 REST API

### 주요 기능 구현
- **성능 최적화**: `Promise.all`을 사용한 병렬 퀴즈 로딩
- **퀴즈 큐 시스템**: 네트워크 지연 없는 부드러운 게임 플레이
- **타이머 관리**: 정밀한 시간 제어
- **애니메이션**: Confetti 효과와 부드러운 전환
- **디바운싱**: 중복 클릭 방지

## 🚀 빠른 시작

### 사전 요구사항

- **Node.js 18+** 및 **pnpm** 설치
- **Quiz API 서버** (포트 12233에서 실행 중)

### 설치 및 실행

```bash
# 저장소 클론
git clone https://github.com/junotech-labs/game-web.git
cd game-web

# 의존성 설치
pnpm install

# 개발 서버 실행
pnpm dev
```

브라우저에서 [http://localhost:5173](http://localhost:5173)을 열어 게임을 즐기세요!

### 프로덕션 빌드

```bash
# 프로덕션 빌드
pnpm build

# 빌드 결과 미리보기
pnpm preview

# Granite을 사용한 배포
pnpm deploy
```

## 🔌 API 연동

### 엔드포인트

```
GET  /quizzes/random  - 랜덤 퀴즈 조회
POST /quizzes/answer  - 답변 제출 및 채점
```

### 환경 변수 설정

`.env.development` 또는 `.env.production` 파일에서 API URL을 설정할 수 있습니다:

```env
VITE_API_URL=http://localhost:12233
```

## 🎨 주요 기능

### ⚡ 성능 최적화
게임 시작 시 `Promise.all`을 사용하여 10개의 퀴즈를 병렬로 로드합니다.
퀴즈 큐 시스템으로 네트워크 지연 없이 부드러운 게임 플레이를 제공합니다.

### 🎯 정확한 타이머
각 문제당 10초의 제한 시간을 정밀하게 관리합니다.
답변 제출 시 타이머가 자동으로 정지되어 정확한 시간 측정이 가능합니다.

### 🎊 시각적 피드백
- **Confetti 애니메이션**: 정답 시 축하 효과
- **진행 표시 타이머**: 남은 시간을 시각적으로 표시
- **카테고리별 색상**: 각 카테고리마다 고유한 색상
- **Kahoot 스타일**: 컬러풀한 답변 버튼

### 🚫 안정적인 UX
- **디바운싱**: 중복 클릭 방지
- **자동 전환**: 정답 후 2.5초, 오답 후 3초 뒤 자동 이동
- **레이아웃 안정성**: 고정된 컨테이너로 화면 떨림 방지
